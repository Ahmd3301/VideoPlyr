# VideoPlyr 🎬

Modern, premium-designed Android video player built using **Media3 ExoPlayer** and **Kotlin**, styled with dimezis **BlurView** glassmorphism controls inspired by `plyr.io`.

## Features
* 📺 Seamless **ExoPlayer** HLS, DASH, and progressive playback.
* 🕶️ Glassmorphic player controls with smooth animations.
* 🔎 Automatic link extraction (`###ex`) via hidden WebView scrapers.
* 🔗 Custom Deep Linking support (`videoplyrio://open` and `videoplyrio://playlist`).
* ⚙️ Bottom Sheet Dialog for playback speed and video quality selectors.
* 🖼️ Picture-in-Picture (PiP) mode support.
